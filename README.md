# Sales Dashboard

A Python project built using **Pandas** and **Matplotlib** to analyze sales data and visualize important business insights.

The project reads a CSV dataset, performs sales analysis, and generates a dashboard with multiple charts to help understand business performance.

---

## Features

- Calculate Total Revenue
- Find the Most Expensive Product
- Find the Cheapest Product
- Identify the Best Selling Product
- Find the Highest Revenue Product
- Calculate Category-wise Revenue
- Calculate Average Product Price
- Display Top 3 Revenue Products
- Identify the Highest Revenue Category

---

## Dashboard Visualizations

The dashboard contains six visualizations:

- 📊 Revenue by Category
- 🥧 Revenue Contribution by Category
- 📊 Top 5 Revenue Products
- 📈 Product Price Comparison
- 📊 Quantity Sold by Category
- 🔵 Revenue vs Quantity (Scatter Plot)

## How to Run

1. Install the required libraries

```bash
pip install pandas matplotlib
```

2. Place `sales.csv` inside the project folder.

3. Run

```bash
python main.py
```
